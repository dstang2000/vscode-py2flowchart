# py2flowchart
Convert python code to flowchart.

## How to use
> Tip: `F1`, `flowchart`
> right-click, 'open flowchart'
![how to use](media/py2flowchart.gif)

## Requirements
- python.
- pip install py2flowchart

## Extension Settings
```json
    "py2flowchart.style": {
        "line-color": "red",
        "element-color": "brown",
        "fill": "yellow",
        "flowstate": {
            "past": {
                "fill": "#CCCCFF",
                "font-size": 12
            }
        }
    },
```
